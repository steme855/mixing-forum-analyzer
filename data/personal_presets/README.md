# Persoenliche Presets

Dieser Ordner ist fuer nutzereigene Preset-Bibliotheken vorgesehen.

- Runtime-Dateien: `<user_id>.json`
- Schreibpfad: `src/personal_presets/PersonalPresetLibrary`
- Zweck: persoenliche EQ-, Dynamics-, Saturation- und Effekt-Presets
- Roadmap: spaetere DAW/VST-Synchronisierung

Die JSON-Dateien werden nicht versioniert, weil sie nutzer- und
projektbezogene Einstellungen enthalten koennen.
