# Feedback-Daten

Dieser Ordner ist fuer laufendes Nutzerfeedback vorgesehen.

- Runtime-Datei: `feedback.jsonl`
- Schreibpfad: `src/feedback/FeedbackStore`
- Zweck: Qualitaetsauswertung fuer Suche, Presets und Summary-Ausgaben

Die JSONL-Dateien werden nicht versioniert, weil sie nutzer- und
sessionspezifische Daten enthalten koennen.
